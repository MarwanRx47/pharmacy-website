# Pharmacy Website

A simple Express + MongoDB pharmacy website starter.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Make sure MongoDB is running locally.
3. Start the app:
   ```bash
   npm run dev
   ```
4. Open http://localhost:3000

## Notes

- The app uses EJS templates in `views/`.
- Add brands and ingredients through MongoDB or by expanding the admin UI.
- The first admin can be created via `/auth/register`.
